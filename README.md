# pdf-to-audiobook

Project to convert a PDF given by the user to an audiobook.

Tested on 64-bit AMD64 CPU with 16GB RAM

Python version:
3.12.1

Install Pytorch:
https://pytorch.org/

TTS:
https://huggingface.co/suno/bark

Before running `python3 tts`:
>>> python3
>>> from huggingface_hub import login
>>> login()
